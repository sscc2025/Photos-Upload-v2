# Photo Upload Sample Page

This is a minimal Node.js + Express project that serves a static HTML/CSS page approximating the sample image provided.

## Getting started (Windows PowerShell)

1. Install dependencies:

```powershell
npm install
```

2. Start the server:

```powershell
npm start
```

3. Open your browser at http://localhost:3000

## Files created
- `server.js` — Express server that serves `public/`.
- `public/index.html` — Sample page HTML.
- `public/styles.css` — Styling to approximate the design.
- `package.json` — Minimal manifest (depends on Express).

If you want a different port, set the `PORT` environment variable before running.
